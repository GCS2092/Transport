# Installation des dépendances pour Leaflet

Exécutez ces commandes dans le dossier frontend :

```bash
npm install leaflet react-leaflet
npm install -D @types/leaflet
```

Ces packages permettent :
- `leaflet` : Bibliothèque de cartes interactives
- `react-leaflet` : Composants React pour Leaflet
- `@types/leaflet` : Types TypeScript pour Leaflet
